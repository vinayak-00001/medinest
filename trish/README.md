# Our Little Scrapbook

A simple one-page romantic scrapbook website.

## Customize

- Edit the text content in `script.js`.
- Replace the placeholder files in `assets/photos/` with your real photos.
- Keep the same file names, or update the `image` paths in `script.js`.

## Run

Open `index.html` in a browser, or start a quick local server:

```bash
python3 -m http.server 8000
```
